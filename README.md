# Online Retail Data Analysis & Segmentation 📊

This project provides a comprehensive analysis of an e-commerce dataset containing over 500,000 transactions.

## 🚀 Key Features
* **Data Cleaning:** Processed missing values and handled duplicates.
* **Exploratory Data Analysis (EDA):** Visualized top-performing countries (UK, Netherlands, Germany) and product trends.
* **Customer Segmentation:** Implemented RFM logic to categorize customers into **Active**, **At Risk**, and **Churned**.
* **API Integration:** Automated currency exchange rate retrieval for financial accuracy.

## 🛠️ Tools Used
* **Python** (Pandas, Matplotlib, Seaborn, Requests)
* **Kaggle** (Development Environment)
* **ExchangeRate-API**
